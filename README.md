
# Rotunda Family Blog 📝

A simple, clean blog site created for the Rotunda family to write and share their thoughts, stories, and updates. Built with React on the frontend and Firebase on the backend (WIP).

---

## 🚧 Project Status

This project is currently **in progress**. The frontend is being built using React, with plans to host using Vercel and to integrate Firebase for:

- 🔥 Firestore (for blog post storage)
- 🔐 Firebase Auth (optional login for post creation)

---

## 🎯 Project Goals

- A minimal, accessible design tailored for ease of use
- A responsive layout that works on desktop and mobile
- A post editor interface for creating and managing content
- Smooth integration with Firebase for data storage and hosting

---

## 📦 Tech Stack

- **React** – Frontend framework
- **Firebase** – Backend-as-a-Service (Firestore, Auth)
- **Vercel** – Hosting
- **React Router** – Page routing
- **(Planned)** Markdown support for post formatting
- **(Optional)** Tailwind CSS or custom styles for clean UI
